# rotating-equipment-pdm
End-to-end predictive maintenance system for industrial rotating equipment using anomaly detection and risk modeling.


## Project Overview
This project demonstrates an end-to-end predictive maintenance system
for industrial rotating equipment (e.g., pumps, compressors).

It covers:
- Sensor data ingestion
- Feature engineering
- Anomaly detection
- Failure risk modeling
- Monitoring and dashboarding
